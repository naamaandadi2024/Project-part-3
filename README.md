# Project-part-3

מערכת לחיזוי דירוג סרטים מבוססת למידת מכונה.

### 1. תיאור הפרויקט
מערכת המקבלת נתוני סרט ומפיקה תחזית לדירוג צפוי באמצעות מודל מותאם.

### 2. הוראות התקנה
יש לפתוח טרמינל בתיקיית הפרויקט ולהריץ:
* יצירת סביבה וירטואלית: `python -m venv venv`
* הפעלת הסביבה:
  * Windows: `venv\Scripts\activate`
  * Mac/Linux: `source venv/bin/activate`
* התקנת ספריות: `pip install -r requirements.txt`

### 3. הוראות הפעלה
הפעלת השרת:
`python api.py`

### 4. גישה לאפליקציה
לאחר הפעלת השרת, ניתן לגשת לאפליקציה דרך הדפדפן בכתובת המופיעה בטרמינל (לרוב: `http://127.0.0.1:5000`).

### 5. שדות קלט
* **Language:** שפת הסרט (ללא ספרות).
* **startYear:** שנה (4 ספרות).
* **runtimeMinutes:** משך בדקות (מספר).
* **genres:** ז'אנרים (טקסט ללא ספרות).
* **Country:** מדינה (טקסט ללא ספרות).
* **plot:** תקציר עלילה.
* **lead_actors_ids:** מזהי שחקנים (מופרדים בפסיקים).

### 6. צוות הפרויקט
נעמה קולומבוס
